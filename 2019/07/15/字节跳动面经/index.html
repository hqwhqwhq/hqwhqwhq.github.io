一面

面了1个小时

  1. 自我介绍
  2. 计算机网络常规问题
    • TCP和UDP区别？TCP保证有序，基本基础的东西都问了
    • TCP三次握手，四次挥手以及相对应的状态
    • TIME_WAIT处于哪个阶段？怎么替代TIME_WAIT（不会）
  3. IO复用
    • epoll有什么优势？（答出来两点）
    • epoll的两种工作模式？（只知道名字）
  4. 计算机操作系统
    • 进程和线程的区别
    • 进程通信方式？线程需要通信吗？
    • 线程安全？（只答出来原子操作，锁）知道cas吗？（不知道）
    • 常用的Linux命令？
    • top命令中load字段代表的含义？（瞎说了一下，不是很清楚）
  5. 数据库
    • MySQL搜索引擎？（InnoDB，MyISAM）还有吗？？？
    • InnoDB优点是什么？（瞎说了速度快）
    • B+树和B树区别？B+树结构画一下？B+树如何保存信息？
  6. C++
    • 堆栈，代码段，静态区这些在内存地址中如何分布？（其实还是没搞懂意思，说了一堆对象内存模型的东西）
    • C++多态的实现？（虚函数，虚表这些说了一堆）
  7. 写道题
    • Trie树（无BUG很快写出来了）
    • 代码有什么地方可以改进？（线程安全，可扩展性方面说了一下）

二面

面的时间比较短，大概就半个小时

  1. 自我介绍
  2. 介绍项目
    • 在阿里做了什么？（趁机把团队的故障演练平台介绍了一下）
    • 在头条做了什么？问我有没有自己实现web IDE的definitions的算法，只说了修改开源分析器的上层部门，没有涉及到修改算法
  3. 写道题
    • 写一个快排+链表？
    • 写得很快，不过最后还是没DEBUG完，太挫了
    • 写一下快排平均时间复杂度下的时间复杂度递推公式？

三面待补充

一开始说要面1个小时左右，后来只面了45分钟

  1. 在阿里做了什么比较有挑战的东西
    • 回答了天天写义务，没有太多挑战的东西。之前设计了一张数据库的连接表，比较有点意思
  2. 括号匹配，*可以代表一个左括号或者一个右括号或者空字符，问给定的字符串合不合法？
    • 贪心，* 号代表右括号尽量匹配右侧，代表左括号尽量匹配左侧
  3. 找一个二叉树最长路径
    • 先找以任意一个节点为根节点的最大深度
    • 再以这个节点为根节点搜索最大深度即为所求
  4. 继承和接口区别
    • 回答了大概就是接口实现多态，工厂模式这种，继承实现相似的实体类
  5. 常用的设计模式
    • 单例，工厂，装饰器
    • 介绍装饰器模式，说成了STL中的配接器模式
